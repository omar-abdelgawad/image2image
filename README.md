# img2img
Seminar project Unsupervised Image-to-Image translation using GANs


## Installation/Usage for contributors
```bash
 $ git clone https://github.com/omar-abdelgawad/image2image.git
 $ cd image2image/
 $ virtualenv venv
 $ source venv/bin/activate
 $ pip install -e ".[dev]"
```
